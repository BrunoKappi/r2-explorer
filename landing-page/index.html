Core Features

Everything You Need to
Manage Cloudflare R2

A complete suite of tools designed to give you full control over your R2 storage buckets — all in a beautiful, intuitive interface.

Recursive Folder Upload

Drag entire folder structures from your OS directly into the explorer. Nested sub-folders are recursively processed, perfectly reproducing your directory tree in R2.

  • External OS Drag & Drop
  • Recursive folder tree replication
  • Floating real-time progress queue
  • Transfer speed indicator
  • Error retry support

Internal Drag & Drop

Reorganize files and folders inside the explorer by dragging rows onto target folders — completely separated from OS file-drop events.

  • Move files between folders visually
  • Isotropic event handling
  • No OS drag interference
  • Visual drop target highlighting

Bulk Actions & ZIP Downloads

Select multiple files or entire folders and perform simultaneous operations. Package items into a dynamically generated ZIP archive processed server-side.

  • Multi-select with Ctrl/Shift/Cmd
  • Marquee lasso selection box
  • ZIP download (server-side streaming)
  • Mass move & delete
  • Copy paths to clipboard

6 View Layout Modes

Switch between six distinct view modes to browse your files exactly the way you prefer — from detailed lists to large icon grids and mosaic previews.

  • Details (list) view
  • Small / Medium / Large icons
  • Extra-Large icons
  • Mosaic view (image gallery)

Rich Context Menu

Right-click any item for a full set of operations. The context-aware menu adapts to files, folders, and empty areas — exposing only relevant actions.

  • Open / Navigate
  • Rename with inline input
  • Move to any directory
  • Copy path / Share CDN link
  • Download file or folder as ZIP
  • View properties & metadata
  • Delete with confirmation

Object Properties & Metadata

Inspect full file metadata including MIME type, MD5 hash (ETag), file size, timestamps, and custom Cloudflare metadata tags — with inline media previews.

  • MIME type detection
  • MD5 / ETag hash display
  • Image & video inline preview
  • Custom metadata tags (key/value)
  • CDN public URL copy

5-Language Internationalization

The entire interface is fully localized in 5 languages. Browser language is auto-detected. Switch on-the-fly using the globe selector.

  • 🇺🇸 English
  • 🇧🇷 Português (pt-BR)
  • 🇪🇸 Español
  • 🇩🇪 Deutsch
  • 🇫🇷 Français

Global Search & Navigation

Search across the entire bucket with a debounced query that filters results reactively. Navigate directories with interactive breadcrumb paths.

  • Full-bucket global search
  • Debounced reactive filtering
  • Interactive breadcrumb navigation
  • Navigate to search result folder

Bucket Statistics

Get a comprehensive overview of your bucket: total files, folder count, cumulative size, average file size, estimated ZIP payload, and file type distribution.

  • Total files & folder count
  • Cumulative storage size
  • Average file size
  • Extension distribution chart
  • Estimated ZIP payload size

Password Protection & Security

All credentials stay strictly server-side. Access is protected by a configurable password gate. CORS origins are configurable per deployment.

  • Server-side credentials only
  • Access password gate
  • Configurable CORS origins
  • Session persistence via localStorage

Keyboard Shortcuts

Navigate and manage files at the speed of thought with built-in keyboard shortcuts that mimic native OS file explorer behavior.

  • Ctrl+A — Select All
  • Ctrl+C / Ctrl+V — Copy & Paste
  • Delete — Delete selected
  • F2 — Rename
  • Enter — Open / Preview

Prefix Mode Toggle

Toggle between virtual folder mode (S3 delimiter navigation) and flat object listing — showing all raw S3 keys recursively without grouping.

  • Virtual folder tree (S3 delimiter)
  • Flat recursive object listing
  • Auto-create bucket on first load

Quick Setup

Get Started in Minutes

Clone, configure, and launch — your R2 bucket manager is ready.

01

Clone & Install

Clone the repository and install all dependencies with npm.

bash 
# Clone the repository
git clone https://github.com/BrunoKappi/r2-explorer.git
cd r2-explorer
npm install
02

Configure Environment

Create a .env file with your Cloudflare R2 credentials.

.env 
R2_ACCESS_KEY_ID=your_access_key
R2_SECRET_ACCESS_KEY=your_secret_key
R2_ENDPOINT=https://ACCOUNT_ID.r2.cloudflarestorage.com
R2_BUCKET_NAME=my-bucket
ACCESS_PASSWORD=your_secure_password
ALLOWED_ORIGINS=http://localhost:5173
03

Launch Dev Server

Start both Express backend and Vite frontend simultaneously.

bash 
# Start frontend + backend concurrently
npm run dev
# Opens at http://localhost:3000
